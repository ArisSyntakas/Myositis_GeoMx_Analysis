# Myositis_GeoMx_Analysis
R code to perform basic analysis on the file structure exported from the GeoMx DSP platform

Basic R packages required: standR, limma, clusterProfiler

Content of Rmd files:

1. GeoMx_standR_limma.Rmd -> Load files exported from the GeoMx DSP, convert to SPE R object, run DGE analysis with limma and create normalized count tables
